# Mine Clicker: Cookie!

📖 This README is also available in [Русский](README.ru.md)

## 🎮 About the Game

In this game, you click on a cookie to earn resources, then buy familiar creatures who help you earn even faster.

## 🚦 Project Status

🟩🟩🟩🟩🟩 — Completed

## 🛠 Tools Used

- Unity (6000.0.37f1)  
- DOTween  
- Zenject  
- Addressables  
- Plugin Your Games  
- Localization
