# Mine Clicker: Cookie!

📖 Этот README также доступен на [English](README.md)

## 🎮 Об игре

В этой игре нужно кликать на печеньку, чтобы зарабатывать ресурсы, а затем покупать знакомых существ, которые помогают зарабатывать ещё быстрее.

## 🚦 Статус проекта

🟩🟩🟩🟩🟩 — Завершён 

## 🛠 Использованные инструменты

- Unity (6000.0.37f1)
- DOTween
- Zenject
- Addressables
- Plugin Your Games
- Localization
